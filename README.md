# 🍽️ EMBER DINING — Premium Restaurant Landing Page

A cinematic landing page for a fine dining restaurant built with modern web technologies.

## Tech Stack
- React 19
- TypeScript
- Tailwind CSS
- GSAP (ScrollTrigger animations)
- Vite

## Features
- Glassmorphic navbar
- Full-screen hero with moody atmosphere
- Signature dishes showcase
- Interactive menu with tab switching
- Reservation form
- Smooth scroll animations
